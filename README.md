# Tutoriel GitHub pour les Débutants

Ce tutoriel est destiné aux néophytes de GitHub. Il vous permettra de comprendre comment créer votre compte GitHub, comment l'optimiser, et comment initialiser ou dupliquer un dépôt (repository).

---

## 1. Création et Configuration de votre compte GitHub

Commençons par la création de votre compte et la personnalisation de votre profil.

### 1.1. Recherche et Accès au Site

Ouvrez votre navigateur web et recherchez "github.com" sur Google, ou accédez directement au site via l'URL.

<img width="521" alt="GitHub - Recherche sur Google" src="https://github.com/user-attachments/assets/2926492b-ad23-42cc-96f6-2c31cea5dfe5" />

Cliquez sur le lien officiel GitHub pour accéder à la plateforme.

### 1.2. Démarrage de l'Inscription

Sur la page d'accueil de GitHub, vous verrez une section pour commencer votre inscription.

<img width="943" alt="GitHub - Homepage" src="https://github.com/user-attachments/assets/33e165fe-090c-4d9b-a214-ff0bf848d506" />

Entrez votre adresse email et cliquez sur "Sign up for GitHub" ou "Sign up" en haut à droite.

### 1.3. Remplir les Informations du Compte

Vous serez redirigé(e) vers le formulaire de création de compte. Remplissez les champs requis (Email, Mot de passe, Nom d'utilisateur, Pays/Région).

<img width="848" alt="GitHub - Inscription" src="https://github.com/user-attachments/assets/41b1a0b7-0b2a-43d6-9eeb-dc6ceb7151e7" />

Vous pouvez décocher la case si vous ne souhaitez pas recevoir de mises à jour de produits. Cliquez ensuite sur "Continue".

### 1.4. Vérification de votre compte (Human Verification - Captcha)

Pour des raisons de sécurité, GitHub vous demandera de prouver que vous n'êtes pas un robot en résolvant une énigme visuelle.

<img width="461" alt="GitHub - vérification par mail (contenu de l'email)" src="https://github.com/user-attachments/assets/038ae082-4d41-4452-b4a8-39eac2dce397" />

<img width="452" alt="GitHub - vérification par Captcha" src="https://github.com/user-attachments/assets/9afef6aa-0d3c-49e8-9f39-6bd4a92dce1e" />

Suivez attentivement les étapes de l'énigme pour prouver que vous êtes une personne.

Une fois l'énigme résolue, cliquez sur le bouton pour valider (par exemple, "Envoyer" ou "Vérifier").

### 1.5. Confirmation de l'adresse email

GitHub vous enverra un code de vérification à l'adresse email que vous avez fournie.

<img width="439" alt="GitHub - Code recu par mail (page de saisie)" src="https://github.com/user-attachments/assets/48db456e-ef74-4f32-881e-3cc5a83d7dbf" />

Ouvrez votre boîte de réception, trouvez l'email de GitHub contenant le code de lancement.

Copiez le code à 8 chiffres et collez-le dans les champs sur la page de GitHub. Cliquez ensuite sur "Continue".

### 1.6. Connexion à votre nouveau compte

Une fois votre email confirmé, votre compte est créé ! Vous serez invité(e) à vous connecter.

<img width="211" alt="GitHub - Connexion" src="https://github.com/user-attachments/assets/9cb82276-7b75-4564-aeca-8a675f8f2eee" />

Utilisez votre nom d'utilisateur et le mot de passe que vous avez définis pour vous connecter. Cliquez sur "Sign in".

### 1.7. Mise à jour de votre profil GitHub (Optionnel)

<img width="938" alt="GitHub - Suivre les étapes 1" src="https://github.com/user-attachments/assets/21a30521-b73f-4437-af78-c564aaac2fe5" />
<img width="890" alt="GitHub - Suivre les étapes 3" src="https://github.com/user-attachments/assets/d37d245e-dd63-49ff-b5c9-791d86d9f567" />

Il est recommandé de personnaliser votre profil pour présenter qui vous êtes sur GitHub. Cliquez sur votre photo de profil (ou l'icône de silhouette) en haut à droite, puis sur "Settings" (Paramètres). Dans le menu latéral, cliquez sur "Public profile".

<img width="754" alt="GitHub - Maj son profil (profil vide)" src="https://github.com/user-attachments/assets/62824a0e-d5a5-48d1-94ff-b5616f20f5b6" />

Remplissez les informations de votre profil (Nom, Email public, Bio, etc.).

<img width="666" alt="GitHub - Maj son profil 2 (profil rempli)" src="https://github.com/user-attachments/assets/331b9dee-9de2-47b9-b94b-6013f2cf8e54" />

Cliquez sur "Update profile" en bas de la page pour sauvegarder vos changements.

---

## 2. Premiers pas avec les Repository GitHub

Maintenant que votre compte est configuré, explorons les dépôts.

### 2.1. Création de votre premier Repository (Dépôt)

Félicitations, vous êtes connecté(e) à GitHub ! Sur votre tableau de bord, vous verrez une option pour créer un nouveau repository.

<img width="526" alt="GitHub - créer son premier repository" src="https://github.com/user-attachments/assets/c33ad485-ce05-47a8-a998-1a80f74ebbbe" />

Cliquez sur "Create repository" pour commencer. Donnez un nom, une description et choisissez les options (public/privé, ajout d'un README).

<img width="656" alt="GitHub - Interface repository (exemple de dépôt)" src="https://github.com/user-attachments/assets/fe2405a1-6566-480c-9e45-0f28879e9d8f" />
*(Note : Cette image montre l'interface générale d'un repository. Sur la page de création, vous remplirez les détails de votre nouveau repository.)*

Cliquez sur "Create repository" en bas de la page. Votre premier dépôt est créé !

### 2.2. Dupliquer un dépôt existant (Fork)

Si vous souhaitez obtenir une copie d'un dépôt qui ne vous appartient pas (par exemple, pour y contribuer ou pour le modifier sans affecter l'original), vous pouvez le "forker".

Allez sur la page du dépôt que vous souhaitez dupliquer (par exemple, `jeremy-jouffroy/school-V4`). En haut à droite, cliquez sur le bouton "Fork".

<img width="705" alt="GitHub - fork (copier un autre repository)" src="https://github.com/user-attachments/assets/b3c5cf53-99e9-493c-9ead-907650ac2fea" />

Confirmez l'opération, et une copie de ce dépôt sera créée sous votre propre compte GitHub.
